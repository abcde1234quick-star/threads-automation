# Writing Algorithm

## 投稿構成4ステップ

1. 1行目フック（損失回避 / 自己開示 / 驚き反転 / 共感問いかけ）
2. 中盤に転換1回（「でも」「ただ」「逆に」で始まる文）
3. 体験・仮説として書く（断定禁止）
4. 末尾はコメント余白（CTA禁止・余韻で終える）

## 文字数（TYPE別上限）

| TYPE | 最適 | 絶対上限 |
|------|------|---------|
| TYPE_C（共感） | 60〜110字 | 130字 |
| TYPE_B（検証） | 90〜140字 | 160字 |
| TYPE_A（比較） | 110〜155字 | 180字 |
| セルフリプライ（全TYPE共通） | 40〜70字 | 85字 |

## 1投稿1論点

1件の投稿に含める主張・比較軸・結論は1つだけ。
情報量より読了率を優先する。
